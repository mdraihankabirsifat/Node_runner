# Node Runner release checklist

Complete this checklist on the final Windows machine and again after uploading to itch.io.

- [ ] Render URL opens successfully: https://node-runner-xayv.onrender.com
- [ ] `/health` responds successfully: https://node-runner-xayv.onrender.com/health
- [ ] Play with bots works
- [ ] Host room works
- [ ] Join room works from another browser or device
- [ ] Music and sound work
- [ ] Keyboard controls work
- [ ] Fullscreen works with F11 and Alt+Enter; Escape exits fullscreen
- [ ] Portable EXE launches without installation
- [ ] ZIP extracts and the included application launches
- [ ] Windows Firewall behaviour checked
- [ ] Itch.io upload downloaded and retested
- [ ] GitHub repository is public
- [ ] No source changes were made after the jam deadline
- [ ] Gameplay video link added
- [ ] Asset licences manually reviewed

Also confirm that the Windows download is described as online-only and unsigned, and that no `.env`, credentials, tokens, Git metadata, tests, or server source appear in the ZIP.
