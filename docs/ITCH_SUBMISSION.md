# itch.io submission guide

Use the Windows x64 ZIP generated by `npm run build:win`. Do not upload the repository source archive as the Windows game.

1. Open [itch.io](https://itch.io/) and create or edit the Node Runner game project.
2. Set **Kind of project** to **Downloadable**.
3. Set pricing to **$0 or Donate** / free with no required payment unless the owner deliberately chooses another option.
4. Upload `dist/Node-Runner-1.0.0-Windows-x64.zip`.
5. Mark that upload as **Windows**.
6. State prominently that the Windows game requires an internet connection.
7. Add `https://node-runner-xayv.onrender.com` as the browser version link in the description.
8. Add the public GitHub repository: `https://github.com/mdraihankabirsifat/Node_runner`.
9. Add the team name and every team member. Do not leave the placeholders from `ITCH_PAGE_COPY.md` in the published page.
10. State the framework: **JavaScript, HTML5 Canvas, Node.js, Express, Socket.IO and Electron**.
11. Add the controls and concise how-to-play instructions from `ITCH_PAGE_COPY.md`.
12. Set **Visibility & access** to **Public** when the final page is ready.
13. Submit the project to the **IUT 12th ICT FEST 2026 GameJam** using the jam's submission page.
14. Confirm that Node Runner appears under the jam's **Submissions** tab.
15. Download the uploaded ZIP from itch.io, extract it into a new folder, launch the included executable, and retest online play.

Before submission, complete [RELEASE_CHECKLIST.md](RELEASE_CHECKLIST.md). Review asset authorship and licences manually; this repository does not provide enough verified information to invent credits safely.
